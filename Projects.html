Simulation Study on Non-Parametric LMPR Tests for Scale

This is a group project in contribution with Saransh Gupta and Rohan P. Shinde. We have extensively performed simulation to study the behaviour of the Locally Most Powerful Rank Tests for Scale Parameter. Additionally, we have derived Test Statistics for specific distributions as well as new Non Parametric tests to test the significance of the scale parameter by improvising on the already existing tests for location parameters.

Biting Fly Data Analysis (Supervised ML)

This is a course project with collaboration with Aditya Negi and harsh Gupta. The goal of this project was to practically apply the cocepts of Multivariate Anaysis on a small dataset. We used classical ML algorithms like LDA, QDA, Logistic Regression to classify the two groups of fly based on their biological characteristics.
